# A/B Testing Analysis – Conversion Rate Optimization

📌 Objective
To determine whether a new product version (Group B) improves user conversion rates compared to the existing version (Group A).

---
Dataset
- Simulated dataset of 1000 users
- Group A: Control group
- Group B: Treatment group

---
Methodology
- Calculated conversion rates for both groups
- Performed hypothesis testing using Z-test
- Evaluated statistical significance

---
Results
- Group B showed a higher conversion rate than Group A
- Statistical testing was used to validate whether the difference is significant

---

Conclusion
The analysis demonstrates how A/B testing helps make data-driven product decisions.  
Such methods are widely used in product analytics to improve user experience and business performance.

Tools Used
- Python
- Pandas
- NumPy
- Statsmodels
